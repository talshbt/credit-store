
##  Client
1. Open new terminal and  Run `cd .\client\`
2. Run `npm i`
3. Run `npm run build`
##  Server
1. Open new terminal 
1. Run `npm i`
3. Run `npm run start` for a  server.

 Navigate to `http://localhost:3080/`. The app will automatically reload if you change any of the source files.
